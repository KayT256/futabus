# FUTA App Clone: Enhanced Customer Experience Proposal

This repo contains a conceptual clone of the FUTA transportation app, featuring a proposed "Trust & Feedback Ecosystem." It was developed as a project for the Marketing Technology class: **"Thiết kế trải nghiệm khách hàng"** (Customer Experience Design).

**TL;DR:** The core feature allows users to click on their booked ticket and view the real-time "Crew Score" (driver/staff ratings and verified tags) aggregated from thousands of previous passengers.

## Key Functional Modules

The proposed solution operates on three core data layers:

* **Public Trust Layer:** Displays an authentic "Crew Score" on the customer interface.
* **Customer Feedback Layer:** A low-friction, quick-response survey system designed to collect raw feedback naturally without annoying the user.
* **Internal Quality Dashboard:** An admin panel for FUTA management to track quality trends, reward top performers, and identify retraining needs.

## UI Highlights

The prototype focuses on three main screens:

1. **Trip Detail:** Displays the assigned driver/staff portrait, name, and Crew Score (e.g., 4.8/5) alongside verified positive tags like "Safe driving," "Enthusiastic," or "Punctual."
2. **Post-Trip Feedback:** Delivered after arrival. Uses quick-select components allowing customers to complete the review in under 20 seconds.
3. **Management Dashboard:** Features charts showing score fluctuations, detailed information, and auto-generates lists of outstanding staff for rewards.
